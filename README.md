### Run the app locally

1- Clone the app
```Shell
git clone https://github.com/persapiens/improve.git
```

2- Build the app
```Shell
mvn clean install
```

3- Run the app
```Shell
java -jar target/improve-1.0-SNAPSHOT.jar
```

4- Access the app with your browser at **http://localhost:8080**
